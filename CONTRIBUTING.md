# Contributing to Kita

Thanks for your interest in contributing to Kita! Kita is a free, open-source running club management app built for Special Olympics and inclusive running clubs. It helps coaches, caregivers, and athletes coordinate training — replacing group chats with a purpose-built tool.

## Getting Started

1. Clone the repo:
   ```bash
   git clone https://github.com/rajivjc/runsosg.git
   cd runsosg
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Copy the environment template and fill in your values:
   ```bash
   cp .env.example .env.local
   ```

4. Start the dev server:
   ```bash
   npm run dev
   ```

5. Open [http://localhost:3000](http://localhost:3000)

## Reporting Bugs

Please open a [GitHub Issue](https://github.com/rajivjc/runsosg/issues) with:

- A clear description of the problem
- Steps to reproduce
- Expected vs actual behavior
- Screenshots if applicable

## Submitting Changes

1. Fork the repo
2. Create a feature branch (`git checkout -b my-feature`)
3. Make your changes
4. Run `npm run lint` and `npm test` to verify nothing is broken
5. Commit with a clear message
6. Open a pull request against `main`
